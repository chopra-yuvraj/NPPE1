## 1. Introduction

Chest X-Ray is one of the most frequently used tools for diagnosis. They can be employed for the diagnosis of various medical conditions from pneumonia and fluid collection to structural abnormalities in the heart and lungs. The sheer volume of X-ray images produced daily by large healthcare facilities makes it practically impossible for radiologists to devote the same level of attention to each one. In addition, the noise from inter-reader diagnosis is unavoidable.

This project aims to address this challenge to take an image of a chest X-ray and classify it into one of 20 thoracic pathology classes:

> Atelectasis, Cardiomegaly, Consolidation, Edema, Effusion, Emphysema, Fibrosis, Hernia, Infiltration, Mass, Nodule, Pleural Thickening, Pneumonia, Pneumothorax, Pneumoperitoneum, Pneumomediastinum, Subcutaneous Emphysema, Tortuous Aorta, Calcification of the Aorta, No Finding

What makes this problem different from the regular classification problem is the cost function. The cost of missing a disease is five times the cost of a false alarm. The macro-averaged asymmetric cost metric is defined as:

| Prediction Outcome | Score |
|---|---|
| True Positive (TP) correctly detected disease | +1 |
| False Positive (FP) alarm with no disease | −1 |
| False Negative (FN) disease missed | −5 |

This asymmetry reflects real world scenario in which failing to catch a disease is far worse than an unnecessary additional scan.

## 2. Methodology

The dataset has 51,043 training images and 17,015 test images, with labels provided in one-hot encoded format (one `1` per row across 20 class columns).

**Class distribution in training data:**

| Class | Count | % of Dataset |
|---|---|---|
| No Finding | 34,079 | 66.8% |
| Infiltration | 5,206 | 10.2% |
| Atelectasis | 2,351 | 4.6% |
| Effusion | 2,156 | 4.2% |
| Nodule | 1,527 | 3.0% |
| Mass | 1,249 | 2.4% |
| Pneumothorax | 1,114 | 2.2% |
| Cardiomegaly | 600 | 1.2% |
| Pleural Thickening | 608 | 1.2% |
| Consolidation | 651 | 1.3% |
| Fibrosis | 389 | 0.8% |
| Edema | 326 | 0.6% |
| Tortuous Aorta | 254 | 0.5% |
| Emphysema | 172 | 0.3% |
| Pneumonia | 160 | 0.3% |
| Calcification of the Aorta | 91 | 0.2% |
| Pneumoperitoneum | 44 | 0.1% |
| Hernia | 37 | 0.1% |
| Subcutaneous Emphysema | 24 | 0.05% |
| Pneumomediastinum | 5 | 0.01% |

This is an extreme imbalance. "No Finding" dominates 66% of the training data. 
Rare classes like Pneumomediastinum (5 instances) and Subcutaneous Emphysema (24 instances) provide the model with almost nothing to learn from.


Several strategies were explored across experiments to address the heavy skew:

- **Class Weighted Cross Entropy Loss**  gives more weight to the loss for minority classes according to the inverse proportion of their frequencies. This forces the network to take the rare classes more seriously.
- **Focal Loss** decreases the loss for examples where the network is already certain and focuses more on the examples the network is unsure about. This is useful if the majority class is consistently easy to predict.
- **Oversampling minority classes** repeats the samples from the minority classes for the network to see more often.

All images were resized to a fixed resolution, either 224x224 or 384x384 depending on the model, and then normalized with the mean and standard deviation of the ImageNet dataset, as well as augmented during training. Validation used center crops of the images without augmentation.


A variety of architectures were implemented and submitted for evaluation:

**Convolutional Networks:**
- **ResNet-18** the baseline. A lightweight network that is fast to train but has a limited field of view for global image context.
- **DenseNet-121**  a popular choice for medical imaging tasks. Dense connectivity between layers retains fine information necessary for identifying pathology.
- **EfficientNet-B4** compound scaling for depth, width, and resolution. More efficient for the number of parameters compared to ResNet variants.
- **ConvNeXt-Base** a modernized version of the pure convolutional network. It incorporates design principles from transformers, including larger kernel sizes and fewer normalization layers.

**Vision Transformers:**
- **ViT-B/16 (Vision Transformer Base)** the standard ViT model for 16x16 patches. This model excels at global context information, which is crucial for chest x-ray imaging where pathology can appear anywhere on the image.
- **Swin Transformer** a hierarchical vision transformer model using a window approach. This approach is more space-efficient than the standard ViT model.
- **Swin-Large Heavy** the large variant of the Swin transformer model with increased augmentation and training.
- **ViT Large Heavy** the large-scale vision transformer model with high augmentation, learning rate warmup, and cosine learning rate schedule. This is the best-performing model overall.

## 3. Experiments

A total of nine experiments were conducted, ranging from large-scale Vision Transformers to a basic CNN baseline. Model architecture, image resolution, augmentation intensity, loss function, learning rate schedule, and training duration were all varied in each experiment.

**Experiment 1 - ResNet-18 baseline:** ResNet-18 with standard cross-entropy loss was used in the initial functional submission. served mainly to verify the submission format and data pipeline. An anomalous score (−7.38918) was obtained in a different run using a larger dataset and a more aggressive pipeline, indicating a label mismatch or format error rather than a model failure.

**Experiment 2 - EfficientNet-B4:** Switched to EfficientNet-B4 for  its compound scaling of depth, width, and resolution. The No Finding majority was countered by applying class-weighted loss. 32 batches, 1e-4 learning rate, and 20 epochs.

**Experiment 3 - DenseNet-121:** because of its dense skip connections that preserve fine-grained features, DenseNet-121 is a common option in medical imaging literature. trained with oversampling of minority classes and focal loss.

**Experiment 4 - ConvNeXt-Base:** presented an updated convolutional architecture featuring GELU activations, layer normalisation, and depthwise convolutions. Pictures were resized to 224 x 224. used warm-up and cosine annealing. 

**Experiment 5 - Swin Transformer (base):** first experiment using a transformer. shifted-window attention with a resolution of 224 x 224. 25 epochs, batch size 32, and learning rate 3e-4 with cosine decay. Colour jitter, rotation ±10°, and random horizontal flips were applied.

**Experiment 6 - Swin-Large Heavy:** Scaled up the Swin architecture to the large variant. Increased augmentation intensity (stronger jitter, random erasing) and extended training to 40 epochs. Validation split: 90/10 stratified by class.

**Experiment 7 - ViT-B/16:** Switched to a plain Vision Transformer using 16×16 patches. Input resolution 224×224. Pretrained ImageNet-21k weights. AdamW optimizer, learning rate warmup over 5 epochs, then cosine decay.

**Experiment 8 - Swin Transformer (re-run with tuned hyperparameters):** Revisited the Swin base architecture with adjusted learning rate (2e-4) and balanced batch sampling to ensure each batch contained at least one sample from each rare class.

**Experiment 9 - ViT Large Heavy:** Largest experiment. ViT-Large with 384×384 input resolution, heavy augmentation (RandAugment, Mixup, CutMix), class-weighted cross-entropy, AdamW with linear warmup and cosine annealing, 35 epochs. This was the final and best-performing submission.

**Validation strategy:**  Throughout, a stratified 90/10 train/validation split was employed to guarantee that uncommon classes were present in both splits. The external validation signal was the public leaderboard score (65% of the test set).

## 4. Results

| Model | Public Score |
|---|---|
| **ViT Large Heavy** | **−4.51195** |
| ViT-B/16 | −4.58870 |
| Swin Transformer | −4.61158 |
| ConvNeXt-Base | −4.62527 |
| Swin-Large Heavy | −4.64258 |
| DenseNet-121 | −4.65060 |
| EfficientNet-B4 | −4.70125 |
| TEST (initial) | −4.72537 |
| ResNet18 + large dataset + optimized pipeline | −7.38918 |

**Vision Transformers outperformed CNNs.** Top performance came from the ViT Large Heavy model. Because chest X-rays depend on broad patterns how lung areas relate to heart silhouette and central structures this kind of network excels by analyzing everything at once.

**Model size alone does not determine rank.** Swin-Large Heavy performed below the compact Swin Transformer base. Because bigger architectures demand precise adjustments especially in regularization and learning rate planning they risk misalignment if tuned poorly. Smaller versions often escape such pitfalls naturally.

**ConvNeXt is competitive.** Fully built on convolutions, yet the Base version (−4.62527) beats DenseNet-121 and EfficientNet-B4. Closing in on transformer-driven designs feels notable here. Modern touches like depthwise convolutions, layer normalization, GELU activation appear useful for this specific job.

**The ResNet-18 experiment was an outlier.** Though built on a strong setup - large data, tuned workflow - the value dropped to −7.38918. That depth of failure hints at something broken in how inputs moved through the system. While models can underperform, they rarely crash this hard. Most likely, labels got scrambled or the output missed required structure.

**The asymmetric cost metric rewards recall over precision.** Because FN costs 5× more than FP, models that predict more cautiously (leaning toward positive predictions when uncertain) generally score better. This was addressed through class weighting and threshold tuning at inference time.

**Score progression:**

```
Start     EfficientNet   DenseNet    Swin-L     ConvNeXt    Swin-T    ViT-B     ViT-L
−4.72537 →  −4.70125  →  −4.65060 → −4.64258 → −4.62527 → −4.61158 → −4.58870 → −4.51195
```

The trajectory shows a consistent upward trend as more capable architectures were adopted, with the transformer family pulling ahead of CNNs in the later experiments.

## 5. Conclusion

A score of −4.51195 emerged as top outcome, delivered by a sizable Vision Transformer using strong data augmentation, class-balanced loss function, alongside cosine decay for learning rate. From conducting all trials, certain patterns became visible

What happens across the full image plays a role when reading chest X-rays. Because some conditions affect several areas at once, seeing connections matters - ViTs handle this naturally through their attention layers. In contrast, CNNs struggle slightly more, given their limited view at each step. Their design forces deeper stacking just to reach similar levels of awareness.

The biggest issue lies in uneven data distribution. Though class weighting made a difference, gains remain limited when facing such skew. Most scoring losses trace back to underrepresented groups. Just five instances of Pneumomediastinum appear alongside two-thirds labeled “No Finding.” Performance gaps persist despite adjustments. Any model struggles when few examples define key cases.

What gets measured drives behavior. Because one missed case equals five incorrect alarms, chasing overall precision misses the point. Adjusting models requires recognizing this imbalance shaping both error penalties and decision rules accordingly. The cost difference demands different math behind the scenes.
